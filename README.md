# School Information System - Frontend

A modern, elegant frontend for a school information system with a traditional touch. Built with React, TypeScript, and Tailwind CSS.

## Design Features

- **Modern with Traditional Hints**: Clean, contemporary design with classic typography and elegant details
- **Neutral Color Palette**: White, black, and gold color scheme for a sophisticated look
- **Smooth Animations**: Framer Motion animations for fluid user experience
- **Responsive Design**: Works seamlessly across all device sizes

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- React Router
- Framer Motion
- Lucide React (Icons)

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will be available at `http://localhost:5173`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Project Structure

```
src/
├── components/       # Reusable components (Layout, Sidebar, Header)
├── pages/           # Page components (Dashboard, Students, Courses, etc.)
├── App.tsx          # Main app component with routing
├── main.tsx         # Entry point
└── index.css        # Global styles with Tailwind
```

## Pages

- **Dashboard**: Overview with statistics and recent activities
- **Students**: Student management and records
- **Courses**: Course catalog and management
- **Grades**: Grade tracking and management
- **Attendance**: Attendance records and tracking
- **Teachers**: Faculty and staff management

## Color Scheme

- **White**: Primary background
- **Black/Charcoal**: Text and borders (charcoal-50 to charcoal-900)
- **Gold**: Accent color for highlights and interactive elements (gold-50 to gold-900)

## Features

- Smooth page transitions
- Hover effects and micro-interactions
- Responsive navigation sidebar
- Search functionality
- Modern table designs
- Card-based layouts
- Elegant typography with serif headings
